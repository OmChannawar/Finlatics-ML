# Finlatics Machine Learning Projects

This repository contains my work from the **Finlatics Machine Learning Course**, where I learned the fundamentals of **Python programming, data preprocessing, and core machine learning algorithms** through hands-on projects.

The course combined **theoretical concepts with practical implementation**, requiring students to code alongside the lessons and apply machine learning techniques to real-world datasets.

---

## Course Overview

The program introduced key concepts required to begin working with machine learning, including:

- Python programming for data analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Training and evaluating machine learning models
- Extracting meaningful insights from datasets

During the course, several fundamental machine learning algorithms were studied and implemented:

- Linear Regression  
- K-Nearest Neighbours (KNN)  
- K-Means Clustering  
- Decision Trees  
- Random Forests  

---

## Projects

After completing the theoretical modules, the course included **three practical machine learning projects** designed to apply the concepts learned.

For each project:

- A **dataset** was provided.
- A **documentation file** described the dataset and defined analytical questions.
- The task was to **clean the data, preprocess it, train machine learning models, and analyse the results**.
- A **short analytical report** was prepared summarising the insights derived from the data.

These projects reinforced the **complete machine learning workflow**, including data preprocessing, model training, evaluation, and interpretation of results.
